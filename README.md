## 快速开始

### 必需依赖
- **AdvancedSlimePaper (ASP)** - 地图实例化
- **QualityArmory** - 武器系统
- **Citizens + Sentinel** - NPC系统
- **ItemsAdder** - 自定义物品
- **SlimeWorldManager** - 地图管理

### 基本命令
- `/exfil` - 打开主菜单
- `/exfil stash` - 打开仓库
- `/exfil secure` - 打开安全箱
